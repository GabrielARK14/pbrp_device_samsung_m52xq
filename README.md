# TWRP Device Tree for Samsung Galaxy A52s 5G

The Galaxy A52s 5G (codenamed _"a52sxq"_) is an upper-mid-range smartphone from Samsung.

It was announced in August 2021 and released in September 2021.

There are also two different variants for Taiwan (a52sxqzt) and Korea (a52sxqks).

## How to build

This device tree was tested and is fully compatible with [minimal-manifest-twrp](https://github.com/minimal-manifest-twrp/platform_manifest_twrp_aosp).

1. Set up the build environment following the instructions [here](https://github.com/minimal-manifest-twrp/platform_manifest_twrp_aosp/blob/twrp-11/README.md#getting-started)

2. In the root folder of the fetched repo, clone the device tree:

```bash
git clone https://github.com/BlackMesa123/android_device_samsung_a52sxq.git -b android-11 device/samsung/a52sxq
```

3. To build:

```bash
export ALLOW_MISSING_DEPENDENCIES=true
. build/envsetup.sh
lunch twrp_a52sxq-eng
mka recoveryimage
```
